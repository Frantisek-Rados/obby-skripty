# 🎮 Obby Game - Roblox Skripty

Zbierka skriptov pre moju **Obby hru** v Roblox Studiu.

---

## 📦 Obsah

### 🔥 Prekážky
| Skript | Popis |
|--------|-------|
| `okamžitá_smrť_pri_dotyku.lua` | Časť, ktorá pri dotyku okamžite zabije hráča |
| `padajúca_plošinka_s_akceleračným_blikaním.lua` | Plošinka, ktorá bliká, zrýchľuje a spadne |
| `plošina_ktorá_sa_točí_rôznymi_smermi.lua` | Točiaca sa plošina |
| `točiaci_sa_laser.lua` | Laser na stĺpe, točí sa ako vrtuľa |
| `hojdacia_plošina.lua` | Plošina, ktorá sa kýve ako hojdačka |
| `posuvná_stena.lua` | Stena, ktorá sa posúva dopredu a dozadu |
| `teleport.lua` | Prenesie hráča na iné miesto |
| `skákacia_podložka.lua` | Vystrelí hráča do vzduchu |
| `padajuce_kamene.lua` | Kamene padajúce zhora |
| `miznuca_plosina.lua` | Plošina, ktorá sa objavuje a mizne |
| `kyvadlo.lua` | Kyvadlo hojdajúce sa zo stropu |
| `padajuca_lava.lua` | Láva padajúca zhora s ohňom |
| `drtic.lua` | Dve steny, ktoré sa približujú k sebe |

### 🎯 Systémy
| Skript | Popis |
|--------|-------|
| `checkpoint.lua` | Ukladá pozíciu hráča |
| `respawn_manager.lua` | Vráti hráča na checkpoint po smrti |
| `casovac_sekcie.lua` | Spustí časovač pri vstupe do zóny |
| `casovac_sekcie_ciel.lua` | Zastaví časovač pri cieli |

### 🪙 Mince a GUI
| Skript | Popis |
|--------|-------|
| `minca_na_zbieranie.lua` | Minca, ktorá sa točí a dá sa zobrať |
| `gui_pocitadlo_minci.lua` | Zobrazuje počet mincí na obrazovke |
| `gui_casovac.lua` | Zobrazuje časovač na obrazovke |

---

## 🛠️ Ako použiť

1. Otvor **Roblox Studio**
2. Klikni pravým na Part → **Insert Object** → **Script**
3. Skopíruj kód z `.lua` súboru do skriptu
4. Spusti hru (**F5**) a otestuj
